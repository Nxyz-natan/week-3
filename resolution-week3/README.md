# to-do list api 
to-do list api with fastapi
## Usage
Go to http://127.0.0.1:8000/docs#/ 
click one of the 5 routes 
get task - lists them
get/search task - searches for them
post task - crethem with priorityate 
patch a task - marks 
or deelte - deltes them
